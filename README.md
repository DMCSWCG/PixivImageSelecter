# PixivImageSelecter V2.0

 Pixiv 搜索指定Tag脚本 
 >功能
 * 加入了GUI界面
 * 使用selenium框架进行半自动登陆(默认FireFox浏览器)
 * 支持屏蔽或者搜索多个Tag
 * 支持指定订阅数、喜欢数和画廊浏览量
 >正在开发(<s>能用就行</s>)
 * 界面刷新BUG修复
 * 只保存ID模式
 * 多线程
 * 其他
